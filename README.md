# Targetomoter

## Overview
Targetomoter is a dissertation-backed prototype for a goal-setting and behavioural productivity system.

The project was developed as part of my MSc dissertation and explores how simple behavioural logic, motivational feedback, and progress tracking can be combined to support user consistency and long-term goal achievement.

## Problem
Many productivity and goal-setting tools allow users to define tasks, but they often lack personalised motivational elements and simple behavioural support mechanisms that help users stay engaged over time.

This project was designed to explore a prototype solution to that problem.

## Project Aim
The aim of this project was to prototype a system that could:
- simulate user goal progress
- provide motivational feedback based on progress and time elapsed
- visualise goal completion trends
- support simple interaction logging for future behavioural analysis

## Repository Structure
- `code/` – prototype logic and demonstration script
- `docs/` – dissertation and supporting academic material
- `requirements.txt` – Python dependencies

## Features
The prototype demonstrates:
- goal progress simulation over time
- rule-based motivational message generation
- progress visualisation with simple plots
- mock backend-style logging functions for:
  - user check-ins
  - goal updates
  - feedback responses

## Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib

## Purpose
This repository is intended to present the underlying prototype logic supporting the wider Targetomoter concept.

It demonstrates early-stage system design thinking in the context of:
- behavioural productivity tools
- progress tracking systems
- motivation support mechanisms
- digital interventions for user goal attainment

## Future Work
