# Targetomoter – Goal Tracking Prototype

## Overview
This project is based on my MSc dissertation, where I designed a goal-setting and productivity application aimed at improving motivation, consistency, and user progress tracking.

The project explores how simple logic, data tracking, and behavioural feedback can support users in achieving their goals.

## Project Structure
- `code/` – Contains prototype Python logic for simulation and tracking
- `docs/` – Contains dissertation and research materials

## Features Demonstrated
- Goal progress simulation
- Motivational feedback logic
- Progress visualisation
- Basic backend-style logging functions

## Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib

## Purpose
This repository demonstrates early-stage system design and prototype logic for a behavioural productivity application.

It reflects my interest in:
- AI-driven behaviour tracking
- productivity systems
- health and lifestyle optimisation tools

## Future Improvements
- Real database integration
- Machine learning for personalised recommendations
- Full app development (mobile/web)
- Real-time user tracking
